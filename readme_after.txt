## 🚀 Démarrage rapide

1. **Lancez l’application** depuis le raccourci sur le Bureau ou le menu Démarrer.
2. Suivez les instructions à l’écran pour scanner un QR code Wi-Fi ou générer un QR code pour votre réseau.
3. Pour toute connexion automatique, assurez-vous d’exécuter l’application en tant qu’administrateur (Windows).

## 📖 Documentation rapide

- Pour plus de détails, consultez le fichier `README.md` ou la documentation en ligne.
- Formats supportés : PNG, JPG, JPEG, GIF, BMP, WebP
- Fonctionnalités principales :
	- Scan d’image ou via webcam
	- Connexion Wi-Fi automatique (Windows)
	- Interface moderne et intuitive

## 💡 Conseils d’utilisation

- Placez le QR code bien en face de la webcam pour une détection optimale.
- Si la connexion échoue, vérifiez le mot de passe et le type de sécurité du réseau.
- Les logs détaillés sont disponibles dans la console de l’application.
